# 🍔 JSON Reader Restaurants

Projeto de estudos para praticar leitura e manipulação de 
arquivos JSON em Python, utilizando dados de redes de fast food.

## 📚 O que foi praticado

- Leitura e parsing de arquivos JSON com Python
- Manipulação e exibição de dados estruturados
- Organização modular com múltiplos arquivos

## 🛠️ Tecnologias

- Python 3
- JSON (nativo do Python)
